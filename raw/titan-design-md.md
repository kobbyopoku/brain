# Titan Design System - Complete Reference

## Color Palette

**Brand Colors:**
- Action Black: #000000
- Highlight Orange: #ff9900

**Neutrals:**
- Midnight Ink: #111111
- Gunmetal Gray: #615e5b
- Soft Concrete: #d8d3cc
- Faded Stone: #e9eaeb
- Off-White Sage: #f3efeb
- Canvas White: #ffffff

## Typography

**Primary Font: Geist**
- Weights: 400, 500, 700
- Sizes: 10–60px (10 values)
- Line height: 1–1.5 (5 values)
- Letter spacing: -0.0300em, 0.0200em, 0.0300em
- Features: 'ss02', 'ss03'
- Fallback: Inter

**Code Font: Geist Mono**
- Weights: 400, 500
- Sizes: 11–48px (5 values)
- Line height: 1.00, 1.10
- Letter spacing: -0.0100em, 0.0300em
- Features: 'ss08'
- Fallback: JetBrains Mono

**Type Scale:**
caption (10px), body-sm (12px), body (14px), body-lg (16px), heading-sm (20px), heading (24px), heading-lg (32px), display-sm (40px), display (60px)

## Spacing & Layout

- Base unit: 4px
- Max width: 1200px
- Section gap: 80px
- Card padding: 28px
- Element gap: 24px

**Spacing Scale:** 4, 8, 12, 16, 20, 24, 28, 32, 36, 40, 52, 56, 60, 64, 80, 88px

**Border Radius:**
- small: 10px
- medium: 20px
- cards: 32px
- navigation: 140px
- buttons: 160px

## Key Design Philosophy

The system employs "high-contrast, monochrome aesthetic" with "generous padding and distinctive pill-shaped radii." Design "feels grounded and authoritative" through "disciplined use of space, typography, and distinct lack of decorative flourishes."

## Critical Guidelines

**Do's:** Use Midnight Ink against Canvas White/Off-White Sage; apply 160px radius to buttons, 140px to navigation; maintain 80px section gaps; use Geist weight 700 at 40–60px for headings with -0.0300em spacing.

**Don'ts:** Avoid additional saturated colors; no box-shadows (use background/border differentiation instead); no generic radii; avoid dense text blocks; no underlines on links.
