# Hyperstudio Design System — Complete Reference

## Color Palette

**Accents:**
- Amber Glow: `#E7C59A` — "Key accent color for interactive elements, 'NEW' tags"
- Neon Green: `#00AC5C` — "Small status indicators, 'spots left' tags"

**Neutrals:**
- Midnight Void: `#101010` — Primary background
- Deep Space: `#080808` — Secondary background
- Polar White: `#F3F3F3` — Primary text
- Absolute Zero: `#FFFFFF` — Accent text/backgrounds
- Ash Gray: `#949494` — Secondary text
- Dark Carbon: `#333333` — Borders
- Slate: `#C1C1C1` — Subtle dividers
- Light Gradients: `linear-gradient(rgb(181, 181, 181), rgb(228, 228, 228))`

## Typography

**Aeonik** (Primary) — weights 400, 700
- Sizes: 13–63px across 10 values
- Line height: 0.95–2.69
- Letter spacing: -0.011, -0.007

**Input** (Monospace) — weight 400
- Sizes: 13–18px
- Line height: 1.2–1.54
- Letter spacing: -0.037, -0.022

**Type Scale:**
- caption: 13px / 1.43
- subheading: 18px / 1.28
- heading-sm: 21px / 1.22
- heading: 23px / 1.11
- heading-lg: 34px / 1.07
- display: 44px / 1.03
- display-lg: 63px / 0.95 / -0.011px

## Spacing & Shape

- Base unit: 4px
- Section gap: 64px
- Card padding: 24px
- Element gap: 10px

**Border Radius:**
- buttons/default: 8px
- tags: 20px
- statusIcons: 99px

## Design Philosophy

The system employs "monochrome terminal with amber accents" aesthetics, emphasizing "high-contrast digital workspace" qualities with minimal color use reserved exclusively for functional indicators.
