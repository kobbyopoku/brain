## Design Philosophy

"Softly lit, expansive workspace" with "pastel gradients and off-white canvas" creating "airy, unburdened atmosphere." Lime green and vivid orange accents reserved for "key actions." Rounded corners on interactive elements juxtapose with "crisp, unadorned typography."

## Do's
- Use Canvas Pearl #faf9f7 for page backgrounds
- Apply Snowdrift White #ffffff for cards/interactive elements
- Reserve Ink Black #000000 for primary text
- Implement Ghost Button styling (1px Ash Border #cfcfcf, 999px radius)
- Utilize Lime Spritz #ade900 for CTA indicators
- Maintain 10px element gap, 45px section gap
- Employ 999px border-radius on primary buttons/tags

## Don'ts
- Avoid saturated background colors; keep gradients pastel
- Don't use dark text on non-light backgrounds
- Don't introduce sharp corners on interactive elements
- Don't apply prominent box shadows (use single subtle shadow only)
- Avoid vivid colors outside Lime Spritz/Sunset Orange
- Don't break established type scale
- Avoid complex background imagery
