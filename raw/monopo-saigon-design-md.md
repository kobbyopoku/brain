# monopo saigon Design System

## Color Palette

**Brand:**
- Deep Ocean Gradient: `linear-gradient(90deg, rgb(160, 224, 171), rgb(255, 172, 46) 50%, rgb(165, 45, 37))`

**Neutrals:**
- Midnight Canvas: `#000000` — Primary background
- Deep Shadow: `#181818` — Secondary text, darker borders
- Misty Gray: `#636363` — Low-contrast interactive elements
- Whisper Gray: `#6d6d6d` — Muted body text
- Frost White: `#ffffff` — Primary text, links, accents

## Typography

**Primary Font — Roobert**
- Weights: 300, 400, 600
- Sizes: 11–225px (12 scale steps)
- Line height: 0.7–1.82
- Role: "Primary brand typeface for all headings, body text, links, and buttons"

**Display Font — Raleway**
- Weight: 400
- Size: 54px
- Line height: 1.39
- Role: "Specific heading elements, providing an alternative, slightly more classic feel"

**Type Scale:**
- Display: 225px, weight 400, 1.25 line height
- 94px, weight 400, 0.76 line height
- 78px, weight 300, 1.1 line height
- Heading-lg: 54px, weight 400, 1.39
- Heading: 39px, weight 400, 1.19
- Heading-sm: 29px, weight 300, 1.58
- Subheading: 18px, weight 400, 1.21
- Body: 16px
- Caption: 11px

## Spacing & Layout

- Base unit: 4px
- Density: Spacious
- Max width: 1078px
- Section gap: 46px
- Card padding: 34px
- Element gap: 14px

**Border Radius:**
- Cards: 10px
- Buttons: 75.024px

## Design Philosophy

"Sophisticated dark aesthetic" employing "organic, shifting gradients" creating "fluid, sculpted surfaces." UI elements are "minimal and translucent." Typography maintains "stark white against dark" with "strong contrast for readability while maintaining understated elegance." Design communicates "depth and hierarchy" through "rich, organic background gradients" rather than traditional shadows.

## Key Guidelines

**Do:**
- Use Roobert as primary typeface with varied weights for hierarchy
- Maintain spacious 14px element gaps
- Apply Deep Ocean Gradient to hero sections
- Use 75.024px radius on all buttons
- Use Frost White text on dark backgrounds

**Don't:**
- Avoid harsh, saturated accent colors
- Never use square borders on interactive elements
- Avoid strong box-shadows or heavy elevation
- Don't deviate from Roobert/Raleway
- Refrain from tight information density
