# Acctual Design System — Complete Reference

## Color Palette

**Brand Accents:**
- Hot Pink: `#f200ca` — Decorative elements, brand highlights in icons
- Vivid Violet: `#6d56fc` — Decorative elements alongside Hot Pink

**Primary Accent:**
- Sky Teal: `#0098f2` — Interactive elements, links, checkmarks, highlights

**Neutrals:**
- Ink Black: `#000000` — Primary text, critical headings
- Graphite: `#0f0f0f` — Prominent headings and body text
- Button Black: `#0d111b` — Primary action buttons
- Deep Slate: `#1e1e1` — Secondary body text and descriptions
- Ash Gray: `#8d8d8d` — Subtle text, metadata, disabled states
- Subtle Cream: `#f7fafc` — Alternative section backgrounds
- Canvas White: `#ffffff` — Page backgrounds, card surfaces

## Typography

**Type Scale (Minor Third 1.2 from 12px base):**
- Display: 64px, weight 600, line-height 1.13
- Heading-lg: 40px, weight 600, line-height 1.2
- Heading: 32px, weight 600, line-height 1.25
- 24px: 24px, weight 600, line-height 1.33
- Subheading: 20px, weight 600, line-height 1.4
- Body: 16px, weight 500, line-height 1.5
- Body-sm: 14px, weight 500, line-height 1.71
- Caption: 12px, weight 400, line-height 1.2

**Font Families:**
- **Open Runde** (primary): weights 400, 500, 600; letter-spacing varies (-0.037em at 64px to normal at smaller sizes)
- **System sans-serif** (secondary): weight 400, 12px; fallback: system-ui, Segoe UI, Roboto, Helvetica Neue, Arial, Noto Sans
- **Caveat** (accents): weight 600, 16px/24px
- **Inter** (alternative body): weight 500, 14px

## Spacing & Shape

**Base Unit:** 4px

**Spacing Scale:** 4, 8, 16, 20, 24, 32, 40, 48, 64, 80, 96, 100px

**Border Radius:**
- Cards: 20px
- Images: 32px
- Buttons: 100px
- Badges: 1250px

**Shadows:**
- Subtle: `rgba(0, 0, 0, 0.06) 0px 2.5px 2.5px 0px`
- Subtle-2: `rgb(36, 38, 40) 0px 0px 0px 1px, rgba(27, 28, 29, 0.48) 0px 1px 2px 0px`

**Layout:**
- Max-width: 1200px
- Section gap: 40–80px
- Card padding: 24px
- Element gap: 4–24px

## Design Guidelines

**Do:**
- Use `#ffffff` Canvas White as dominant background
- Apply Open Runde for headings and body text with varying weights
- Employ 100px radius for buttons and pill-shaped elements
- Reserve Sky Teal exclusively for interactive elements
- Maintain 24px and 40px padding for sections
- Use Caveat for testimonials and impactful statements

**Don't:**
- Introduce colors beyond Sky Teal, Hot Pink, Vivid Violet
- Use box shadows for elevation; prefer subtle background changes
- Use system sans-serif for headlines
- Deviate from letter-spacing values for Open Runde
- Create dense sections without adequate whitespace
- Mix font sizes/weights within single lines
