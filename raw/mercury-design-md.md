# Mercury Design System - Complete Reference

## Color Palette

**Brand Accent:**
- Mercury Blue: #5266eb (Primary CTA buttons)
- Ghost Blue: #cdddff (Secondary button backgrounds, hover states)

**Neutrals:**
- Deep Space: #171721 (Outermost page background)
- Midnight Slate: #1e1e2a (Primary page/section backgrounds)
- Graphite: #272735 (Button backgrounds, interactive surfaces)
- Lead: #70707d (Borders, dividers, UI accents)
- Silver: #c3c3cc (Secondary text, footer copy, disabled states)
- Starlight: #ededf3 (Primary text for headlines, body, navigation)
- Pure White: #ffffff (Text on primary CTA buttons)

## Typography

**Font Families:**
- arcadiaDisplay: Headlines (weights 360, 480, 530; sizes 21–65px)
- arcadia: Body, labels, navigation (weights 360, 400, 420, 480; sizes 12–21px)

## Spacing & Layout

- Base unit: 4px
- Density: spacious
- Page max-width: 1200px
- Section gap: 80–120px
- Element gap: 12–32px

**Spacing scale:** 4px, 8px, 12px, 16px, 20px, 24px, 32px, 40px, 56px, 72px, 80px, 112px, 128px

**Border Radius:**
- Cards: 0px

- Buttons: 32px, 40px
- Containers: 4px

## Design Philosophy

"The design feels like a command center at twilight, expansive and focused. A deep, near-black neutral palette (#1e1e2a, #171721) creates an immersive, cinematic canvas where glowing off-white text (#ededf3) provides crisp clarity."

## Key Guidelines

**Do:**
- Use arcadiaDisplay at light weight (360) for major headlines
- Reserve Mercury Blue exclusively for primary CTAs
- Apply deep neutral palette for immersive environments
- Use extreme corner radii (32–40px) for signature pill shapes
- Maintain high contrast with Starlight text
- Use generous vertical spacing (80px+) between sections
- Differentiate interactive list items with 1px Lead bottom borders

**Don't:**
- Use Mercury Blue for text, backgrounds, or decoration
- Use font weights exceeding 530
- Apply shadows; use color and opacity shifts instead
- Introduce saturated colors beyond monochrome plus blue accent
- Use small corner radii on buttons
- Reserve Pure White for primary CTA text only
- Create dense layouts; prioritize breathing room
