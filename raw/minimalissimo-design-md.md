# Minimalissimo Design System

## Color Palette

| Name | Hex | Purpose |
|------|-----|---------|
| Inkwell | #000000 | Primary text, core UI elements, strong borders, icon fills |
| Sterling | #999999 | Medium-contrast borders, control outlines, structural separators |
| Storm Gray | #a1a1a1 | Tertiary body text, informational text, less prominent details |
| Pale Ash | #e0e0e0 | Neutral form states, badge text, quiet UI feedback |
| Whisper White | #efefef | Background for subtle interactive elements like inputs |
| Canvas | #f5f5f5 | Dominant page and section backgrounds |
| Porcelain | #ffffff | Component backgrounds within sections, elevated surfaces |

## Typography

**Font Family:** GeistSans (Weight: 400 only)
**Fallback:** Inter, Arial, sans-serif
**Scale Ratio:** Minor Second (1.067) from 16px base

| Role | Size | Line Height | Letter Spacing |
|------|------|-------------|----------------|
| caption | 12px | 1.5 | normal |
| body-sm | 14px | 1.43 | normal |
| body | 16px | 1.4 | normal |
| heading | 24px | 1.1 | normal |

## Spacing & Shape

**Base Unit:** 4px
**Density:** comfortable

| Value | Token |
|-------|-------|
| 4px, 8px, 12px, 16px, 24px, 32px, 40px, 48px, 64px, 96px | `--spacing-[4-96]` |

**Section gap:** 96px | **Card padding:** 16px | **Element gap:** 16px

**Border Radius:** Links/inputs/buttons: 4px | Cards: 8px

## Core Philosophy

"White gallery canvas" aesthetic emphasizing "restrained, almost stark" design where "content is paramount" using "achromatic palette with crisp contrasts." Typography is "compact and precise." Components maintain "ghost-like presence, subtly hinting at interaction."
