# claude-mem

> Long-term memory for Claude Code via end-of-session compression — saves the key decisions and context of a session into a persistent store that carries over to the next session.

## Profile

This page is a **stub**. claude-mem appears in this wiki only via [[wiki/sources/regent0x-claude-code-247-dev-team]], which describes it as one of three tools in the persistent-memory layer (alongside an Obsidian vault and [[wiki/entities/claude-subconscious]]). The mechanism described: at the end of each session, claude-mem compresses key decisions and context into a persistent store; the next session reads from that store.
