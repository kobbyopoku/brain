# Next.js

> React-based web framework, published by [[wiki/entities/vercel|Vercel]], used as the default deployment target on the Vercel platform and as a common stack choice in this wiki's AI-tooling sources.

## Profile

Next.js is a React framework for building web applications. In this wiki it appears as a stack component in worked Claude Code setup examples and as a docs target for documentation-freshness tooling. Its tight integration with [[wiki/entities/vercel|Vercel]] (Vercel founder Guillermo Rauch created Next.js) makes it the canonical framework for the platform's deployment flows.
