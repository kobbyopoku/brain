# OpenCode CLI

> Open-source coding-agent CLI — competitor to [[wiki/entities/claude-code|Claude Code]]. Built by SST (Serverless Stack) as an open-source alternative. Listed as one of 15 auto-detected agent CLIs in [[wiki/entities/open-design|Open Design]].

## Profile

This page is a **stub**. OpenCode CLI appears in the wiki via Open Design's enumeration of detectable coding agents. The relevance: OpenCode is one of the few **open-source** coding-agent CLIs in the auto-detect list (most are vendor-published proprietary tools). For users wanting fully OSS workflows, the Open Design + OpenCode CLI combination is end-to-end open-source.

## Key facts

- **Maintainer**: SST team.
- **License**: open-source.
- **Role**: terminal-based coding-agent CLI.
- **Open Design integration**: detected on PATH and invoked as a runtime alternative.
