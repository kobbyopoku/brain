# NVIDIA Dynamo

> Appears in this wiki via [[wiki/sources/theahmadosman-inference-engines-local-ai-hardware]].

## Key facts

- **Type**: Distributed orchestration layer that sits above inference engines such as vLLM, SGLang, and TensorRT-LLM — per [[wiki/sources/theahmadosman-inference-engines-local-ai-hardware]].
- **Capabilities**: Supports disaggregation, intelligent routing, and multi-tier KV caching — per [[wiki/sources/theahmadosman-inference-engines-local-ai-hardware]].
- **Positioning**: Used when single-engine serving is no longer enough; the recommended choice for cluster orchestration — per [[wiki/sources/theahmadosman-inference-engines-local-ai-hardware]].
