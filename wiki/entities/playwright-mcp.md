# Playwright MCP

> An MCP server / Claude Code plugin that lets Claude drive a browser to test flows and validate fixes.

## Profile

Playwright MCP is presented as "Tool 3" in [[wiki/sources/suryanshti777-9-claude-code-plugins-senior-engineer]]'s nine-plugin senior-engineer stack. It gives [[wiki/entities/claude-code]] browser control: opening browsers, clicking buttons, testing flows, inspecting UI issues, reproducing bugs, and validating fixes automatically. The thread frames it as the point where AI coding "starts becoming autonomous."

## Key facts

- **Type**: MCP server / Claude Code plugin. *(per [[wiki/sources/suryanshti777-9-claude-code-plugins-senior-engineer]].)*
- **Capabilities**: open browsers, click buttons, test flows, inspect UI issues, reproduce bugs, validate fixes automatically.
- **Example use**: "Find why the signup flow breaks on mobile."
