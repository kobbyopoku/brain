# tricalt

> X poster and Cognee maintainer arguing that memory and skills are the same agent harness, not separate plugins.

Appears in this wiki via [[wiki/sources/tricalt-memory-skills-same-harness]].

## Key facts

- **Author**: wrote the X post "Memory isn't a plugin. Skills aren't a plugin. They're the same harness." (status 2055876832797581406, 2026-05-17) ([[wiki/sources/tricalt-memory-skills-same-harness]]).
- **Prior post reach**: claims an earlier post arguing "skills aren't static files" got 1.7M views (status 2032179887277060476) ([[wiki/sources/tricalt-memory-skills-same-harness]]).
- **Affiliation**: associated with [[wiki/entities/cognee|Cognee]] ([[wiki/entities/topoteretes|topoteretes]]) and its self-improvement / skill-as-memory-node work ([[wiki/sources/tricalt-memory-skills-same-harness]]).

- Prescribes an Observe -> Inspect -> Amend -> Evaluate loop to keep skills from degrading over time ([[wiki/sources/tricalt-memory-skills-same-harness]]).
