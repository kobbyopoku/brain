# AutoGen

> Microsoft-origin agent framework that pioneered conversation-driven orchestration and is evolving into the Microsoft Agent Framework.

## Profile

AutoGen is an agent framework noted for pioneering conversation-driven orchestration, in which multiple agents coordinate through structured conversation rather than an explicit graph DSL. Per [[wiki/sources/akshay_pachaar-x-anatomy-of-an-agent-harness]], it is evolving into the Microsoft Agent Framework and is built on a three-layer architecture (Core, AgentChat, Extensions) supporting five orchestration patterns. It is also named among agent orchestration frameworks at Layer 10 of the autonomous-agent stack in [[wiki/sources/awrigh01-technical-stack-autonomous-agents]].

## Key facts

- **Origin / direction**: pioneered conversation-driven orchestration; evolving into the **Microsoft Agent Framework** (per [[wiki/sources/akshay_pachaar-x-anatomy-of-an-agent-harness]]).
- **Architecture**: three layers — Core, AgentChat, Extensions (per [[wiki/sources/akshay_pachaar-x-anatomy-of-an-agent-harness]]).
- **Orchestration patterns**: five — sequential, concurrent (fan-out / fan-in), group chat, handoff, and magentic (per [[wiki/sources/akshay_pachaar-x-anatomy-of-an-agent-harness]]).
- **Stack position**: named among agent orchestration frameworks at Layer 10 (per [[wiki/sources/awrigh01-technical-stack-autonomous-agents]]).

## Positions and claims

- **Conversation-driven orchestration is a first-class agent coordination pattern.** *(Implicit in the framework's design, per [[wiki/sources/akshay_pachaar-x-anatomy-of-an-agent-harness]].)*
