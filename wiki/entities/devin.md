# Devin

> Autonomous software-engineering agent by [Cognition Labs](https://www.cognition.ai). Originally launched as a hosted agent that could plan and execute multi-step engineering tasks; now also accessible via CLI. Listed as one of 15 auto-detected agent CLIs in [[wiki/entities/open-design|Open Design]].

## Profile

This page is a **stub**. Devin appears in the wiki via Open Design's enumeration of detectable coding agents. Cognition Labs (Devin's maker) was an early, well-funded entrant in the autonomous-agent category. The Devin CLI is the local-runtime variant of what was originally a hosted SaaS agent.

## Key facts

- **Publisher**: Cognition Labs.
- **Role**: autonomous software-engineering agent — plans, codes, runs, debugs.
- **Distinguishing claim**: positioned as more autonomous than typical coding-agent CLIs (longer-running, less interactive).
- **Open Design integration**: detected on PATH and invoked as a runtime alternative.
