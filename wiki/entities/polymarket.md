# Polymarket

> Prediction-market platform hosting Musk-weekly-tweet-count markets, appearing in this wiki via [[wiki/sources/retrochainer-claude-700k-musk-tweets]].

## Profile

This page is a **stub**. Polymarket appears in the wiki only via the [[wiki/sources/retrochainer-claude-700k-musk-tweets]] ingest. No primary source about Polymarket itself has been ingested.

## Key facts

- Runs markets on real-world events, including [[wiki/entities/elon-musk|Elon Musk]]'s weekly tweet count (per [[wiki/sources/retrochainer-claude-700k-musk-tweets]]).
- Tweet-count markets split into ~30 separate range contracts (e.g. <50, 50–75, 75–100, >200 tweets).
- Each range is a separate contract whose price equals the implied probability (a contract at $0.34 = 34% implied chance); the correct range settles at $1.00.
- Each range has its own order book of buy and sell orders.
- Exposes an API that supports automated 24/7 trading bots.
