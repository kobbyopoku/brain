# MLX-LM

> The LLM package built on [[wiki/entities/mlx|MLX]] — adds Hugging Face Hub integration, quantization, fine-tuning, and distributed inference.

## Profile

MLX-LM is the large language model package built on top of [[wiki/entities/mlx|MLX]]. It layers in Hugging Face Hub integration, quantization, LoRA and full fine-tuning, distributed inference, and access to a large MLX Community model ecosystem. Its bundled server warns that it is not recommended for production because it implements only basic security checks. It appears in this wiki via [[wiki/sources/theahmadosman-inference-engines-local-ai-hardware]].

## Key facts

- **Built on**: [[wiki/entities/mlx|MLX]] (cited to [[wiki/sources/theahmadosman-inference-engines-local-ai-hardware]]).
- **Capabilities**: Hugging Face Hub integration, quantization, LoRA and full fine-tuning, distributed inference, and a large MLX Community model ecosystem ([[wiki/sources/theahmadosman-inference-engines-local-ai-hardware]]).
