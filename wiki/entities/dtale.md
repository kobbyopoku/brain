Appears in this wiki via [[wiki/sources/zostaff-22-hedge-fund-quant-open-source-repos]].

## Key facts

- **Origin**: Open-sourced by Man Group; ~5.1k stars — the most-starred Man Group repo (source-reported) — per [[wiki/sources/zostaff-22-hedge-fund-quant-open-source-repos]].
- **Function**: Provides an Excel-like browser interface over pandas DataFrames (filters, correlations, charts) from one line of code — per [[wiki/sources/zostaff-22-hedge-fund-quant-open-source-repos]].
