# ONNX Runtime GenAI

> An engine that implements the full generative loop over ONNX Runtime and powers Microsoft's local-AI surfaces.

## Profile

ONNX Runtime GenAI implements the full generative loop on top of ONNX Runtime, and underpins several Microsoft local-AI surfaces including Foundry Local, Windows ML, and the VS Code AI Toolkit. It supports a broad range of execution providers and is best suited to app deployment and ONNX-based workflows.

## Key facts

- **Architecture**: implements the full generative loop over ONNX Runtime; powers Foundry Local, Windows ML, and the VS Code AI Toolkit (cited to [[wiki/sources/theahmadosman-inference-engines-local-ai-hardware]]).
- **Execution providers**: supports CPU, CUDA, DirectML, TensorRT-RTX, OpenVINO, QNN, WebGPU, and AMD GPU (cited to [[wiki/sources/theahmadosman-inference-engines-local-ai-hardware]]).
- **Best fit**: app deployment and ONNX workflows (cited to [[wiki/sources/theahmadosman-inference-engines-local-ai-hardware]]).
