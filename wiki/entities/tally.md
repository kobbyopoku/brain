# Tally

> A fictional brand built end-to-end as a demonstration of Anthropic's Claude Design. Appears via [[wiki/sources/nateherk-claude-design-tally-brand]].

- **Fictional demo brand** built end-to-end by nateherk using [[wiki/entities/claude-design|Claude Design]] — pitch deck, landing page, mobile prototype, brand guidelines, and an animated launch video, deployed live to Vercel (cited to [[wiki/sources/nateherk-claude-design-tally-brand]]).
- A **372-line markdown brand spec** for Tally was built in regular Claude before opening Design (cited to [[wiki/sources/nateherk-claude-design-tally-brand]]).
