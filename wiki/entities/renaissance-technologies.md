# Renaissance Technologies

> The most-secretive quant fund, framed in this wiki as the closed-moat archetype with zero public code.

## Profile

Renaissance Technologies is a quantitative hedge fund managing roughly $130B in assets (source-reported), described as averaging around 66% annual returns for three decades. It has zero public GitHub repositories and does not allow former employees to write detailed resumes about their work. In the source's framing, it represents the "edge is the alpha you discover, close everything" archetype — the opposite of firms that open-source tooling. It appears in this wiki via a roundup of hedge-fund and quant-firm open-source repositories.
