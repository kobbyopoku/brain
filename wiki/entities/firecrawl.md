# Firecrawl

> **Agent-optimized web search and scraping**. Per Ole Lehmann's 12-integrations Hermes guide: *"cleaner data, faster responses than the native Hermes search, and it burns way fewer tokens."* If Firecrawl + Browserbase both plugged in, agents pick automatically. Endorsed in the wiki as a Research-job integration in [[wiki/sources/itsolelehmann-hermes-12-integrations|Ole's 4-job model]].
