> File-storage [[mcp-server|MCP integration]] — one of four MCP servers [[wiki/sources/khairallah-ai-automations-10k-month|Khairallah]] names as core competencies for an [[ai-automation-services|AI automation services]] builder.

## Profile

This page is a **stub**. Google Drive appears in this wiki only via [[wiki/sources/khairallah-ai-automations-10k-month]], where it serves as the file-storage destination in the practice exercise (search the web, save a summary to Drive, email it, post to Slack). No primary source about Google Drive's broader product surface is yet ingested.

## Key facts

- **Website**: https://drive.google.com
- **MCP role**: file storage and access.
- **Claude connector role**: as a connector, lets Claude read documents, reference files, and save outputs directly (e.g. the Weekly Report); feeds the Morning Briefing (cited in [[wiki/sources/heynavtoor-personal-ai-system-claude]]).
- Cited (as "Drive") as a tool in a one-command skill workflow (Notion → Gmail → Drive → Output) (cited in [[wiki/sources/charliejhills-full-agent-system-6-steps]]).
- Listed (as "Drive") as a candidate Claude Connector for the two-Connectors setup (cited in [[wiki/sources/zephyr-hg-7-setups-claude-fluency]]).
- In 8xgrowth's plan, the Auto-Post Agent pulls edited videos from Google Drive and pushes them to socials automatically (cited in [[wiki/sources/8xgrowth-100-days-to-10k-clipping]]).
