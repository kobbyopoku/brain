# xai-org/x-algorithm (repository)

> The claimed open-source GitHub repository containing X's For You feed recommendation system, from which [[wiki/sources/doublenickk-personal-x-agent-algorithm]] is built.

Appears in this wiki via [[wiki/sources/doublenickk-personal-x-agent-algorithm]].

## Key facts

- Located at `github.com/xai-org/x-algorithm` per the source (cited to [[wiki/sources/doublenickk-personal-x-agent-algorithm]]).
- Described as containing four main components: Home Mixer (orchestration), Thunder (in-network retrieval), Phoenix (ML ranking model), and a Candidate Pipeline framework.
- Its `phoenix/scoring` code is said to contain the formula Final Score = Σ (weight_i × P(action_i)) over 14 predicted actions.
- Contains an Author Diversity Scorer that attenuates a post's score each time it has already been shown to the same user in a session.
- Existence and contents are a source claim, not independently verified in this ingest.
