# Safe (Safe Wallet)

> Smart-account / multisig wallet whose allowance, Zodiac, and multisig primitives map onto enterprise agent governance.

Appears in this wiki via [[wiki/sources/awrigh01-technical-stack-autonomous-agents]].

## Key facts

- **Allowance modules**: provide caps per token per spender, one-time or recurring (cited to [[wiki/sources/awrigh01-technical-stack-autonomous-agents]]).
- **Zodiac modules**: role-based modules and hierarchical multisig with security councils and timelocks map onto enterprise agent governance (cited to [[wiki/sources/awrigh01-technical-stack-autonomous-agents]]).
- **Treasury guidance**: Safe's own documentation recommends a 2-of-4 multisig with human signers for agent-controlled treasuries (cited to [[wiki/sources/awrigh01-technical-stack-autonomous-agents]]).
