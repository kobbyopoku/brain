# Nader Dabit (dabit3)

> Developer + technical writer. Author of the wiki's canonical reference for **agent hooks** — the deterministic-control mechanism that makes agent workflows programmable. Companion GitHub repo: [`dabit3/agent-hooks-in-depth`](https://github.com/dabit3/agent-hooks-in-depth).

## Key contributions

- Articulated the **6-lifecycle-point starter set** for agent hooks: `SessionStart` / `UserPromptSubmit` / `PreToolUse` / `PostToolUse` / `Stop` / `SessionEnd`.
- The **`event → matcher/filter → handler → outcome`** operating model.
- The **load-bearing distinction**: *"Use prompts for guidance. Use hooks for behavior that should run every time."*
