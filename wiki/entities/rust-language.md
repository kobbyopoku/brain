# Rust

> Programming language, appearing in this wiki via [[wiki/sources/akintola-steve-backend-1-million-users]].

Appears in this wiki via [[wiki/sources/akintola-steve-backend-1-million-users]].

## Key facts

- Recommended (with [[wiki/entities/go-language|Go]]) for raw performance and memory efficiency (cited to [[wiki/sources/akintola-steve-backend-1-million-users]]).
- Implementation language for [[wiki/entities/pggraph|pgGraph]]; its CSR traversal runs in a "tight Rust loop" with no garbage collection (cited to [[wiki/sources/daleverett-postgres-as-graph-pggraph]]).
- pgGraph is built as a [[wiki/entities/pgrx|pgrx]] extension, which enables native Postgres extensions written in Rust (cited to [[wiki/sources/daleverett-postgres-as-graph-pggraph]]).
