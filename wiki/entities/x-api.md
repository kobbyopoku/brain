# X API

> The posting and analytics interface for the X (formerly Twitter) platform, used by a personal X agent to publish approved content on schedule and read performance data.

Appears in this wiki via [[wiki/sources/doublenickk-personal-x-agent-algorithm]].

## Key facts

- **Role in the agent build**: used in Blueprint 02 to publish approved content on schedule (per [[wiki/sources/doublenickk-personal-x-agent-algorithm]]).
- **Free tier**: has posting limits per the source (per [[wiki/sources/doublenickk-personal-x-agent-algorithm]]).
- **Elevated access**: Blueprint 03 requires X API elevated access to pull analytics data (per [[wiki/sources/doublenickk-personal-x-agent-algorithm]]).
