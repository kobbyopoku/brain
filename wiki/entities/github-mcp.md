# GitHub MCP

> An MCP server / Claude Code plugin that turns Claude into a GitHub operator with whole-repo understanding.

## Profile

GitHub MCP is presented as "Tool 2" in [[wiki/sources/suryanshti777-9-claude-code-plugins-senior-engineer]]'s nine-plugin senior-engineer stack. It connects [[wiki/entities/claude-code]] to GitHub so the model can read repositories, review pull requests, inspect issues, summarize commits, understand project architecture, and generate fixes from GitHub context — replacing manual file-pasting with whole-repo understanding.

## Key facts

- **Type**: MCP server / Claude Code plugin. *(per [[wiki/sources/suryanshti777-9-claude-code-plugins-senior-engineer]].)*
- **Capabilities**: read repos, review PRs, inspect issues, summarize commits, understand project architecture, generate fixes from GitHub context.
