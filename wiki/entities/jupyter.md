# Jupyter

> Interactive notebook platform underlying several catalogued quant tools and a core developer tool in the AI-engineer roadmap.

Appears in this wiki via [[wiki/sources/zostaff-22-hedge-fund-quant-open-source-repos]] and [[wiki/sources/techwithtimm-ai-engineer-roadmap-2026]].

## Key facts

- **Underlies catalog tools**: `beakerx`, `notebooker`, `pyflyby`, and `nbstripout-fast` build on Jupyter (cited to [[wiki/sources/zostaff-22-hedge-fund-quant-open-source-repos]]).
- **Funding**: D.E. Shaw funded the development of IPython, Jupyter, and NumPy (cited to [[wiki/sources/zostaff-22-hedge-fund-quant-open-source-repos]]).
- **Roadmap role**: Named as a Stage II core developer tool — notebooks for testing ideas, running Python code, and working with data interactively (cited to [[wiki/sources/techwithtimm-ai-engineer-roadmap-2026]]).
