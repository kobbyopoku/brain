# Claude

> Anthropic's AI assistant product, whose consumer surfaces (Projects, Skills, Connectors, scheduled tasks, model picker) are inventoried in this wiki as the building blocks of a "7 setups" fluency workflow.

## Profile

Claude is Anthropic's AI assistant product. Its consumer surfaces include Projects, saved Skills, Connectors, scheduled tasks, and a model picker spanning Haiku, Sonnet, and Opus. It supports a project-root `CLAUDE.md` file declaring voice, audience, constraints, and output rules, and Connectors that wire it to external services. In this wiki it appears as the subject of [[wiki/entities/zephyr-hg|Zephyr]]'s thread on Claude fluency, where these surfaces are catalogued as the seven reusable setups.

## Key facts

- **Vendor**: Anthropic's AI assistant product. (per [[wiki/sources/zephyr-hg-7-setups-claude-fluency]])
- **Consumer surfaces**: Projects, saved Skills, Connectors, scheduled tasks, and a model picker (Haiku / Sonnet / Opus). (per [[wiki/sources/zephyr-hg-7-setups-claude-fluency]])
- **CLAUDE.md**: supports a project-root `CLAUDE.md` file declaring voice, audience, constraints, and output rules. (per [[wiki/sources/zephyr-hg-7-setups-claude-fluency]])
- **Connectors**: connect to Gmail, Calendar, Drive, Slack, Notion, and GitHub. (per [[wiki/sources/zephyr-hg-7-setups-claude-fluency]])
- **Model tiers**: Haiku (fast/cheap/simple), Sonnet (normal work), Opus (hard reasoning). (per [[wiki/sources/zephyr-hg-7-setups-claude-fluency]])
