# Text Generation Inference (TGI)

> Hugging Face's production LLM serving engine.

## Profile

Text Generation Inference (TGI) is Hugging Face's production server for serving large language models. It ships with observability features (tracing, metrics) and performance features (tensor parallelism, continuous batching), and is best suited to deployments where Hugging Face integration and simplicity matter.

## Key facts

- **Features**: production server with tracing, metrics, tensor parallelism, and continuous batching (cited to [[wiki/sources/theahmadosman-inference-engines-local-ai-hardware]]).
- **Best fit**: when Hugging Face integration and simplicity matter (cited to [[wiki/sources/theahmadosman-inference-engines-local-ai-hardware]]).
