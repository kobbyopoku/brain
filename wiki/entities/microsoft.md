> Technology company that, in this wiki, appears as the steward of the Microsoft Agent Framework (AutoGen's successor) and as the publisher of three free GitHub beginner curricula central to an AI-engineer roadmap.

## Profile

This page is a **stub**. Microsoft appears in two ingested sources as an AI-framework steward and an open-source educational publisher. No primary Microsoft source is yet ingested.

## Key facts

- **Microsoft Agent Framework** — AutoGen is evolving into the Microsoft Agent Framework ([[wiki/sources/akshay_pachaar-x-anatomy-of-an-agent-harness]]).
- Publishes `microsoft/generative-ai-for-beginners` — covers prompts, transformers, embeddings, chat apps, and LLM production behavior ([[wiki/sources/Shruti_0810-zero-to-ai-engineer-roadmap]]).

- Named as a frontier-AI knowledge source alongside [[wiki/entities/openai|OpenAI]], [[wiki/entities/anthropic|Anthropic]], [[wiki/entities/google|Google]], and [[wiki/entities/nvidia|NVIDIA]] ([[wiki/sources/Shruti_0810-zero-to-ai-engineer-roadmap]]).
