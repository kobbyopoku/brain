# Thariq Shihipar

> **Lead for Claude Code at Anthropic**, formerly YC W20 + MIT Media Lab. Author of *"Using Claude Code: The Unreasonable Effectiveness of HTML"* — a substantive 2026-05 post arguing HTML beats Markdown as Claude's output format. Also conducted a 2-hour Claude Agent SDK workshop at AI Engineer Summit. Active builder of agentic systems at Anthropic.

## Profile

Thariq is the closest the wiki has to an *internal Anthropic* voice on agentic-AI architecture. His content is unusual in the AI-creator landscape because: (a) he ships the production code that other people build on; (b) he's published meaningful framework arguments (HTML-as-output, prompt-caching-as-everything); (c) he's authored open-source demos (e.g. local email agent built on Claude Agent SDK).

His [examples gallery](https://thariqs.github.io/html-effectiveness) of 20 HTML artifacts is one of the cleanest demonstrations of what *artifact-first agentic output* looks like in practice.

## Key facts

- **Role**: Claude Code lead at Anthropic.
- **Prior**: YC W20 alum, MIT Media Lab, builder/contributor to Claude Code's agent harness.
- **X handle**: [@trq212](https://x.com/trq212).

- **Notable claims**:
  - HTML beats Markdown as agent output format when output is meant for human review/comparison/editing.
  - Long-running agentic products like Claude Code are made feasible by **prompt caching** (reuse computation across roundtrips).
  - The Claude Code SDK was renamed to the Claude Agent SDK because the agent harness proved useful for much more than coding.
  - Anthropic is moving to use the Claude Agent SDK to power most of their own agent loops.

## Positions and claims

- **Handoff > execution as the hard part of agentic work.** A model can finish the task; the human still has to judge risk, split follow-ups, share context, record decisions. Output format is a handoff-quality lever.
- **HTML as a workspace-bridge format.** *"If an output is meant to be reviewed, clicked, compared, edited, or shared, HTML is often closer to the real workspace."*
- **Markdown won the GPT-4-token-budget era; modern context windows make HTML viable again.**
- **Prompt caching is foundational.** Without it, long agentic loops are economically infeasible.
