# Beren Millidge

> Author of the 2023 essay "Scaffolded LLMs as Natural Language Computers," cited in this wiki for the analogy framing an LLM harness as an operating system. Appears via [[wiki/sources/akshay_pachaar-x-anatomy-of-an-agent-harness]].

## Profile

This page is a **stub**. Beren Millidge appears in one ingested source as the originator of the scaffolded-LLMs / Von Neumann analogy. No primary Millidge source is yet ingested.

## Key facts

- Wrote the 2023 essay "Scaffolded LLMs as Natural Language Computers" ([[wiki/sources/akshay_pachaar-x-anatomy-of-an-agent-harness]]).
- Framed a raw LLM as a CPU with no RAM/disk/IO — with the context window as RAM, external databases as disk, tools as device drivers, and the harness as the OS ([[wiki/sources/akshay_pachaar-x-anatomy-of-an-agent-harness]]).
- Wrote "We have reinvented the Von Neumann architecture" ([[wiki/sources/akshay_pachaar-x-anatomy-of-an-agent-harness]]).

## Positions and claims

- An LLM agent's components map onto classical computer architecture, with the harness playing the role of the operating system — argued in "Scaffolded LLMs as Natural Language Computers" ([[wiki/sources/akshay_pachaar-x-anatomy-of-an-agent-harness]]).
