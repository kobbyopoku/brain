# Guri Singh (heygurisingh)

> X content creator. Author of *"How to Set Up Claude Cowork (And Quit ChatGPT for Good)"* — the most comprehensive Claude Cowork onboarding guide yet ingested into the wiki. Provides specific folder structure, file templates, global instructions, day-to-day workflow patterns, and 11 plugins. Targets **knowledge workers** rather than developers — *"if you don't write code, mastering Claude Cowork should be your number one priority right now."*

## Recurring framings

- **Forget about prompts; the game is text files.** *"The more context you give it as files, the fewer prompts you need."*
- **Cowork has real read and write access to the folder you share. If something goes wrong, you want the damage to be contained.** Folder protocol matters.
- **AskUserQuestion as the most-transformative feature.** *"Always add this to your prompts: Start by using the AskUserQuestion tool before responding to gather enough context."* Claude generates an interactive form (buttons, options, rankings).
- **80% of Cowork sessions can start with the same prompt structure**: *"I want [TASK] for [SUCCESS CRITERIA]. First, explore my CLAUDE COWORK folder. Then, ask me questions using the AskUserQuestion tool. I want to refine the approach with you before you execute."*
- **Anthropic launched 11 official plugins for Cowork**: Sales, Marketing, Legal, Finance, Data Analysis, Product Management, Customer Support, etc.
