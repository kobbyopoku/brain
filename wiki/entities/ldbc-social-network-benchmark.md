# LDBC Social Network Benchmark

> Graph benchmark used to report [[wiki/entities/pggraph|pgGraph]]'s traversal latencies, appearing in this wiki via [[wiki/sources/daleverett-postgres-as-graph-pggraph]].

Appears in this wiki via [[wiki/sources/daleverett-postgres-as-graph-pggraph]].

## Key facts

- Benchmark scale cited: 3.1 million nodes and 34.5 million edges (cited to [[wiki/sources/daleverett-postgres-as-graph-pggraph]]).
- Used to report pgGraph hot-path timings: Person search 9.8ms, Friend traversal 34.1ms, Post-to-tag path 6.5ms (cited to [[wiki/sources/daleverett-postgres-as-graph-pggraph]]).
- Basis for the ~34x Neo4j memory comparison reported for pgGraph (cited to [[wiki/sources/daleverett-postgres-as-graph-pggraph]]).
