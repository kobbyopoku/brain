# Sequential Thinking

> An MCP server / Claude Code plugin that forces Claude to break problems into steps and reason through them before answering.

## Profile

Sequential Thinking is presented as "Tool 5" in [[wiki/sources/suryanshti777-9-claude-code-plugins-senior-engineer]]'s nine-plugin senior-engineer stack, where it is called "one of the most underrated tools." It pushes [[wiki/entities/claude-code]] to decompose a problem into steps, reason through architecture, evaluate alternatives, and debug logically rather than jumping to an answer.

- **Type**: MCP server / Claude Code plugin. *(per [[wiki/sources/suryanshti777-9-claude-code-plugins-senior-engineer]].)*
- **Function**: forces step-by-step engineering reasoning — decompose, reason through architecture, evaluate alternatives, debug logically.
- **Author framing**: "one of the most underrated tools."
