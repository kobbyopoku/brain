# Qwen CLI

> Coding-agent CLI built on Alibaba's Qwen (通义千问) model family — typically open-weight Qwen-Coder models. Listed as one of 15 auto-detected agent CLIs in [[wiki/entities/open-design|Open Design]].

## Profile

This page is a **stub**. Qwen CLI appears in the wiki via Open Design's enumeration of detectable coding agents. The Qwen-Coder family of models (Qwen2.5-Coder, Qwen3-Coder) is among the strongest open-weight coding-model lines, making the Qwen CLI a meaningful option for users wanting open-weight alternatives to Claude / GPT.

## Key facts

- **Publisher**: Alibaba (Qwen team).
- **Underlying model**: Qwen-Coder family (open weights typically available on Hugging Face).
- **Role**: terminal-based coding-agent CLI.
- **Open Design integration**: detected on PATH and invoked as a runtime alternative.
