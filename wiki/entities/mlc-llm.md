# MLC LLM

> A compiler-first universal deployment engine for shipping LLMs everywhere.

## Profile

MLC LLM is a compiler-first deployment engine designed to run large language models across a wide range of platforms. It exposes OpenAI-compatible APIs across REST, Python, JavaScript, iOS, and Android, and is geared toward shipping LLMs to browser, mobile, and native app targets (including WebLLM).

## Key facts

- **Architecture**: compiler-first universal deployment engine with OpenAI-compatible APIs across REST, Python, JavaScript, iOS, and Android (cited to [[wiki/sources/theahmadosman-inference-engines-local-ai-hardware]]).
- **Best fit**: shipping LLMs everywhere, especially browser, mobile, and native apps (including WebLLM) (cited to [[wiki/sources/theahmadosman-inference-engines-local-ai-hardware]]).

## Positions and claims

- Emphasizes universal, cross-platform deployment via a compiler-first approach, per [[wiki/sources/theahmadosman-inference-engines-local-ai-hardware]].
