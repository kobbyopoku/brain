# tldraw

> Appears in this wiki via [[wiki/sources/athletickoder-building-agents-first-principles]].

Canvas / diagram editor the source's toy text-to-diagram agent miniaturizes; a "real tldraw agent" is the scaled-up target example.

## Key facts

- **Role in source**: the toy diagram agent is described as "a tiny version of a tldraw-style agent" — cited to [[wiki/sources/athletickoder-building-agents-first-principles]].
- **Scaled-up example**: a real tldraw environment "changes the validator, not the loop": execute JSON actions inside the real editor, let tldraw sanitize and apply them, read back shapes/bindings/selections/errors, and export a screenshot to compute reward — cited to [[wiki/sources/athletickoder-building-agents-first-principles]].
