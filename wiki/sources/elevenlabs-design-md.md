# ElevenLabs Design System (DESIGN.md from Refero)

> *"Architect's blueprint on warm vellum"* — type-first, museum-label aesthetic. **Near-zero saturation**, four typefaces (Waldenburg display 300 only + WaldenburgFH product labels 700 only + Inter body + Geist Mono), and color reserved exclusively for two product avatar dots. The most editorial/restrained system in the catalog.

## TL;DR

ElevenLabs runs the catalog's most chromatically restrained palette: 8 grayscale neutrals + 2 saturated accents (`#ff4704` Ember and `#0447ff` Signal Blue) used **only for ElevenAgents and ElevenCreative avatar dots**. Page ground is Eggshell `#fdfcfc` — never pure white. Cards float on hairline 1px shadows, never depth shadows. *"All UI elements in the same perceptual plane."* Inputs have 0px radius as editorial distinction. Buttons are 9999px pills. Custom Waldenburg display only at 300 weight; the 700-weight WaldenburgFH is reserved for 14px product labels exclusively. Inter never below 13px (product UI) / 14px (marketing prose).

## Key takeaways

- **Color discipline is extreme**: only 2 saturated colors, used only for avatar dots.
- **Two Waldenburg variants**, mutually exclusive in usage: 300-weight for display 32-48px, FH 700-weight for product labels 14px only.
- **Eggshell `#fdfcfc` page ground** — explicit "never pure white" rule.
- **Hairline elevation only**: 1px shadow, no depth layering. Everything in one perceptual plane.
- **Inputs at 0px radius** as a brand signature against the otherwise pill-heavy system. Same convention as Antimetal.
- **Third-party logos desaturated to `#b1b0b0`** (Fog) — explicit rule.
- **Geist Mono 400 13px** only for technical annotations like `[whispers]`, `[sarcastic]` — voice-related metadata.
