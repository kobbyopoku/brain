# Column Design System (DESIGN.md from Refero)

> *"Architectural blueprint on white marble"* — same philosophy phrase as Stripe but a different execution: Deep Plum `#111a4a` instead of Deep Violet, Action Orange `#ec652b` as the primary attention color, multi-typeface system (SuisseIntl primary + SFMono + Inter + SuisseIntlMono).

## TL;DR

Column's system is highly typeface-rich (four named typefaces) with a Deep Plum identity color and an Action Orange call-out color. Seven shadow tokens — most in the catalog. 8px border-radius universally for cards/buttons. Compound shadow stacks (5-layer rgba combinations) for elevated cards. Two semantic colors (Success Moss, Info Blue) explicitly named — most other brands omit semantic colors from their DESIGN.md. The "very light, almost invisible dotted grid background" is part of the philosophy (a literal blueprint metaphor).

## Key takeaways

- **Same philosophy as Stripe** ("architectural blueprint on white marble") — different identity color (`#111a4a` Deep Plum vs Stripe's `#533afd` Deep Violet) and different attention color (`#ec652b` Action Orange).
- **Four typefaces**: SuisseIntl primary + SFMono code + Inter body + SuisseIntlMono. Densely typographic.
- **Seven shadow tokens** — the most explicit shadow system in the catalog.
- **Numerical-data-as-mono**: SFMono and SuisseIntlMono at 10-12px specifically for "numerical data and code." Notable brand-level distinction.
- **Semantic colors named**: Success Moss `#44b48b`, Info Blue `#7ea7e9` — uncommon in DESIGN.md (most omit semantic).
- **Display weights are 500** (not 700 like Apple, not 300 like Stripe). Mid-weight identity.
