# Hyperstudio Design System (DESIGN.md from Refero)

> *"Monochrome terminal with amber accents"* — extreme dark ground (`#101010` Midnight Void / `#080808` Deep Space), Amber Glow `#E7C59A` for interactive accents, Neon Green `#00AC5C` for status indicators only. Aeonik primary + Input monospace. Compact spacing.

## TL;DR

Hyperstudio is the most spartan system in the catalog: 8 neutrals + 2 accents (Amber Glow for interactive elements + 'NEW' tags; Neon Green for small status indicators). Aeonik for display 13-63px (-0.011 letter-spacing); Input monospace for technical text 13-18px. Three border-radius values (8px buttons/default, 20px tags, 99px status icons). Tight 10px element gap, 24px card padding, 64px section gap.

## Key takeaways

- **Two accents only**: Amber Glow `#E7C59A` (interactive), Neon Green `#00AC5C` (status). Highly disciplined.
- **Extremely dark ground**: `#101010` Midnight Void darker than most "dark mode" systems.
- **Three border-radius values total** — minimal vocabulary.
- **No documented shadows** in the captured DESIGN.md — likely no-shadow system.
- **Aeonik + Input** typeface pairing — common in technical/dev-tool brands.
