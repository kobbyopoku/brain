# Ui Design System (DESIGN.md from Refero)

> *"Monochromatic architectural blueprint"* — stark black-and-white aesthetic; Geist + Geist Mono typography (Vercel-associated family); deliberate corner-radius tension between hard 10-14px on cards/inputs and near-pill on buttons/badges.

## TL;DR

The "Ui" entry uses Vercel's Geist family (sans + monospace) for all typography, achromatic palette, very subtle elevation. Distinctive shape choice: hard 10-14px corner radii for layout (cards, inputs) paired with near-pill rounding for action elements (buttons, badges). Single design philosophy: precise, functional, machine-like.

## Key takeaways

- **Geist family**: Geist for primary brand text + Geist Mono for code/monospace. The Geist family is Vercel's open-source typeface — reusable by any product wanting a Vercel-adjacent typographic feel.
- **Achromatic palette**: stark black/white with minimal accent. Almost pure-monochrome.
- **Corner-radius split**: hard 10-14px (cards, inputs, structure) vs near-pill (buttons, badges, action). Geometric tension as identity.
- **Type scale**: 12 / 13 / 14 / 16 / 18 / 48 px. Tight scale with single 48px display jump.
- **Three weights**: 400 / 500 / 600. Functional range.
