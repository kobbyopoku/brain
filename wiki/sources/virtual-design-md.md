# Virtual Design System (DESIGN.md from Refero)

> *"Minimalist digital console"* — deep black backgrounds, crisp white text, accent orange purely for status guidance, white outline borders defining interactive elements. The most restricted weight + size catalog entry.

## TL;DR

The "Virtual" entry is a dark-mode design with maximum discipline: only three type sizes (10/12/16px), only one weight (500), three typefaces with specific-and-narrow roles (neue-haas-grotesk-display for body+nav, IPM for impact headlines, Arial for button subtext). Accent orange is reserved purely for status indicators — no decorative use. Interactive elements defined by white outline borders maintaining minimalist aesthetic.

## Key takeaways

- **Single weight**: 500 only. Most catalog brands have 3-5 weights.
- **Three sizes only**: 10 / 12 / 16 px — joins [[wiki/sources/perplexity-ai-design-md|Perplexity]] for smallest type scale.
- **Three typefaces with narrow roles**: neue-haas-grotesk-display (body, nav), IPM (impact display, tight tracking), Arial (button subtext) — each used for one specific job.
- **Orange-as-status**: accent orange reserved for status/guidance, not decoration.
- **White outline borders**: interactive elements defined by border, not fill — preserves minimalist aesthetic.
- **Console-like mood**: precise control + focused functionality on a near-black background.
