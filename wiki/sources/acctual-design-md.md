# Acctual Design System (DESIGN.md from Refero)

> Pill-shape, multi-typeface, off-white-canvas + Sky-Teal-only-for-interaction system.

## TL;DR

Acctual reserves a single accent color (Sky Teal `#0098f2`) for *all* interactive elements while permitting decorative use of Hot Pink and Vivid Violet in icons. Buttons are 100px-radius pills; badges are 1250px (effectively circular). The type system is unusually multi-font for a small brand — Open Runde primary + Inter alternative + Caveat for testimonials + system sans-serif for captions. Canvas is `#ffffff` with `#f7fafc` for alternate sections; "no box shadows for elevation, prefer subtle background changes."

## Key takeaways

- **Single interactive color**: Sky Teal `#0098f2` exclusively. Hot Pink and Vivid Violet decorative-only.
- **Pill-everything**: 100px radius buttons, 1250px badges. The system reads as "soft" because every interactive shape is round.
- **Four typefaces** is unusual at this scale: Open Runde (display + body) + Inter (alternative body) + Caveat (testimonials/accents) + system sans (captions). Most brands run leaner.
- **Spacing**: 4px base, 24px card padding, 40-80px section gap.
- **No drop shadows**: subtle-2 is a 1px outer ring on `rgba(27,28,29,0.48)`. Elevation via background color shifts only.
