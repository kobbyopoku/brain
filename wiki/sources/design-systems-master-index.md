- **172 DESIGN.md files total** in the canonical `<brand>/DESIGN.md` folder layout (refactored 2026-05-05).

- **10 brands have BOTH** Refero and Open Design DESIGN.md files (overlap section below).

## Path convention

As of 2026-05-05, every design system lives in a brand-named folder with a single `DESIGN.md` file inside:

```
raw/<brand>/DESIGN.md                                  # Refero brands
raw/open-design/design-systems/<brand>/DESIGN.md       # Open Design brands
```

This mirrors the canonical convention used by Open Design's upstream repo and most open-source design-token projects. The pattern makes the path predictable from the brand name alone.
