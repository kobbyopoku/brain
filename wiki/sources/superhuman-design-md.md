# Superhuman Design System (DESIGN.md from Refero)

> *"Cinematic cockpit behind warm light"* — Aubergine `#421d24` + Iris `#714cb6` brand pair, Parchment Canvas `#f2f0eb` ground (never pure white), single proprietary Super Sans VF typeface (weights 460/500/540/600/700). Hero Dusk Gradient as the brand atmosphere.

## TL;DR

Superhuman is single-typeface (Super Sans VF) and aggressively brand-distinct: Aubergine `#421d24` for the announcement banner and footer only, Iris `#714cb6` for ghost button outlines and links, Lavender Chip `#d4c7ff` as a soft button fill. Parchment Canvas `#f2f0eb` ground — *"never pure white."* Glassmorphic panels via backdrop-filter blur(12px). Five-weight variable font (460/500/540/600/700). Display 64px weight 540 with line-height 0.96 and -0.028em tracking — extremely tight.

## Key takeaways

- **Single typeface system** — Super Sans VF carries everything, with five custom-weight variations.
- **Aubergine + Iris** as the brand color pair — unusual aubergine identity.
- **Aubergine usage rule**: announcement banner and footer ONLY. Never mid-page content.
- **Parchment Canvas ground**: warm off-white, never pure white.
- **Iris for outline only**: ghost buttons, links, focus rings. Never as filled button background.
- **Backdrop blur 12px** for glassmorphic panels — same technique as Raycast (much shallower blur).
- **64px display at line-height 0.96**: literally below 1.0 — display lines overlap slightly. Aggressive typographic tightness.
