# All-In-One Salon (GlossGenius) Design System (DESIGN.md from Refero)

> *"Crisp digital ledger, with neon highlights guiding the way"* — GlossGenius's beauty-professional software: ultra-clean Basel Grotesk Book typography, monochromatic palette with vivid yellow-green and pastel violet-blue accents, sharp text balanced by soft rounded UI shapes.

## TL;DR

GlossGenius (cataloged as "All-In-One Salon" in Refero) is sophisticated digital-assistant software for the beauty industry. Two Basel typefaces handle distinct roles — Basel Grotesk Book for UI/body, Basel Classic Book for editorial display headlines. Largely monochromatic palette punctuated by vivid yellow-green and pastel violet-blue accents reflecting the beauty industry's vibrancy without being overwhelming.

## Key takeaways

- **Two-typeface Basel system**: Basel Grotesk Book (UI body) + Basel Classic Book (editorial display) — same family, different role + voice.
- **Sharp/soft balance**: sharp Basel Grotesk Book contrasted with soft, rounded UI shapes — professionalism + user-friendliness.
- **Energetic accent palette**: yellow-green + pastel violet-blue — vibrancy without overwhelming clinical software interfaces.
- **Three weights**: 400 / 500 / 600. Functional restraint.
- **Wide type scale**: 13 / 14 / 16 / 18 / 22 / 32 / 40 / 48 / 72 px — substantial range.

## Notable quotes

> "Crisp digital ledger, with neon highlights guiding the way."

## Notes

- **GlossGenius is the canonical brand name**: the philosophy text identifies the product as GlossGenius, but Refero's catalog title labels it "All-In-One Salon" — likely Refero's category framing.
- **Beauty-industry design conventions**: the palette tension (clinical software needs trust + the beauty industry needs visual energy) is well-resolved here through the *neon-highlight-on-monochrome* strategy. A useful reference point for any clinical-or-professional-services brand needing visual energy.
