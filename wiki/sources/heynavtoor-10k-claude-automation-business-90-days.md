# How to Build a $10K Per Month Claude Automation Business in 90 Days — heynavtoor

> A 6-phase 90-day playbook for building an AI automation services business. **Functionally a derivative** of [[wiki/sources/khairallah-ai-automations-10k-month|Khairallah's 6-phase $10K/month playbook]] — same structure, same niches, same pricing math, same window argument. Useful as a second-source confirmation of the canon, with a few operational deltas worth capturing.

## TL;DR

heynavtoor (whose other thread is [[wiki/sources/heynavtoor-10-repos-replace-eng-team]]) presents the $10K/month AI-automation services playbook condensed into a 90-day timeline. Phase 1 (toolkit, days 1-14) → Phase 2 (niche, 14-21) → Phase 3 (free first case study, 21-30) → Phase 4 (package + price, 30-35) → Phase 5 (paying clients, 35-60) → Phase 6 (scale to $10K/month, 60-90). Pricing math identical to Khairallah: 5 retainer × $750 + 2 builds × $5,000 = $13,750/mo ≈ $165k/yr. Recommended stack: Claude Code or Cowork + Notion/Obsidian + Cal.com + Stripe payment links + Loom. Total stack cost <$200/mo.

## Key takeaways (deltas vs. Khairallah)

The substantive content overlaps ~90% with [[wiki/sources/khairallah-ai-automations-10k-month]]. Worth capturing only what's new:

- **Adds Law firms** as an 8th niche (Khairallah named 7: real estate / marketing agencies / e-commerce / professional services / content creators / financial advisors / healthcare). Law firms specifically noted for: document review, deposition summaries, client intake, billing letters.
- **Names Cowork** as a Claude Code alternative — a new tool entry. Not yet in this wiki; worth a stub on next ingest if cited again.
- **Heavier emphasis on sub-agents** vs. Khairallah's emphasis on skill files. heynavtoor's framing positions sub-agents (not skills) as "the engine." Both are valid lenses on the same Claude Code primitive — different authors privilege different units.
- **Specific recommended stack** (5 tools, ~$200/mo): Claude Code or Cowork + Notion/Obsidian + Cal.com + Stripe + Loom. Khairallah is silent on infrastructure choices; heynavtoor names them.
- **Companion article reference**: "30 Claude Code Sub-Agents I Actually Use" — a paste-ready sub-agent template library. Not yet ingested.
- **Tighter pricing posture**: explicitly says *"Never charge by the hour. It punishes you for being fast."* Khairallah's *"value-based not hours-based"* says the same thing more politely.
- **Specific outreach math**: 100 outreach messages → 10 calls → 2-3 closes. Khairallah is silent on conversion rates.
- **Three-trap list** that kill 90% of attempts: perfectionism on the toolkit (8 weeks studying instead of building), pricing too low (<$3,000 signals amateur), no niche (serving everyone serves no one).
