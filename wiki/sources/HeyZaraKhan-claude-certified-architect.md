# Claude Certified Architect Resource List — HeyZaraKhan

> A consolidated resource list pointing at the [[wiki/entities/claude-certified-architect]] program plus adjacent free Anthropic resources (Skilljar courses, anthropic-cookbook, MCP and API documentation, partner network). Useful as a discovery / pointer source rather than as substantive analysis.

## TL;DR

HeyZaraKhan compiles links for working through Anthropic's Claude Certified Architect program: the access-request form, 13 free Skilljar courses, the anthropic-cookbook GitHub repo, an exam guide, free practice questions at claudecertifications.com, MCP documentation at modelcontextprotocol.io, the API documentation, and the partner network. A community-uploaded post-exam playbook is also linked. Comments note the program is gated to corporate-email registrations (it is positioned for employees of partner companies).
