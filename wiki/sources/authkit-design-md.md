# AuthKit Design System (DESIGN.md from Refero)

> *"Midnight Command Center"* — WorkOS's AuthKit hosted-auth UI: dark frosted-glass surfaces over a `Midnight Abyss` blue-black canvas, restrained muted blues, three custom typefaces (Untitled Sans + aeonikPro + dotDigital) for distinct functional zones.

## TL;DR

AuthKit's design is a polished internal-dashboard feel: deep blue-black background, crisp white text, achromatic muted blues, frosted-glass surfaces with subtle inner shadows. Three custom typefaces with distinct roles — Untitled Sans for body + forms, aeonikPro for prominent display, dotDigital for monospaced numerical/technical text. Deliberate mix of pill-rounded interactive elements and subtly-rounded containers.

## Key takeaways

- **Dark frosted-glass aesthetic**: glowing softly in a dark room; subtle inner shadows; restrained muted blues.
- **Three-typeface system with distinct functional zones**: Untitled Sans (body, forms, buttons), aeonikPro (display + brand voice — wider for visual contrast), dotDigital (monospaced numbers via `tnum` for technical detail).
- **Pill + subtle-round corner mix**: deliberate tactile contrast between fully-pilled interactive elements and gently-rounded containers.
- **Type scale**: 12 / 14 / 16 / 18 / 24 px — narrow, focused.
- **Polished without sterile**: design captures the *internal-dashboard* category — productivity tooling that doesn't feel cold.

## Notable quotes

> "Midnight Command Center. Imagine a high-tech dashboard glowing softly in a dark room, with frosted glass elements reflecting subtle light."

## Notes

- **WorkOS context**: AuthKit is the UI layer of WorkOS's hosted authentication product, used by SaaS teams that don't want to build login pages from scratch.
- **Three-typeface design** is uncommon — most catalog brands use 1-2 typefaces. The functional separation here (display vs body vs monospace numerical) demonstrates a sophisticated typographic system.
