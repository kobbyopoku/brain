# Agent stack

> The full market-infrastructure stack autonomous agents need in order to *transact* — identity, trust, pricing, contracting, settlement, enforcement, and policy — not just the model and workflow layers. Appears in this wiki via [[wiki/sources/awrigh01-technical-stack-autonomous-agents]].

## Definition

This is a **stub**. The agent stack, as framed by [[wiki/sources/awrigh01-technical-stack-autonomous-agents]], is a 3-plane, 10-layer infrastructure for an agent economy: a *trust plane* (identity, discovery, reputation), a *market plane* (quoting, contracting, settlement, dispute resolution), and a *control plane* (governance, compliance, orchestration/runtime). The thesis is that markets run on identity, trust, pricing, contracts, settlement, enforcement, and policy — not on intelligence alone — and that the fragments exist but do not yet compose into a coherent marketplace.

## Treatment across sources

- [[wiki/sources/awrigh01-technical-stack-autonomous-agents]] decomposes the stack into 3 planes (trust / market / control) and 10 layers; argues whoever owns identity, settlement, and governance owns the marketplace for machine labor.
