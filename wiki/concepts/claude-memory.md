# Claude Memory

> Claude's per-account learning system that builds a user profile from conversations, corrections, and repeated mentions over time.

## Definition

Claude Memory is described as Layer 3 of a personal AI system: a per-account feature that learns about a user from their conversations, corrections, and repeated mentions, accumulating a profile that persists across sessions. It is toggled in Settings → Memory, can be manually seeded with "Remember that..." instructions, and is editable and deletable.

## Why it matters

Persistent per-user context is foundational to the "personal AI system" framing the wiki tracks, and to Godwin's interest in agentic systems that retain state across sessions. Memory is the layer that makes Claude "know" a user rather than starting cold each time.
