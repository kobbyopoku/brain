# Compounding judgement

> Appears in this wiki via [[wiki/sources/petradonka-agents-need-feedback-loops]]: the goal of an agent feedback loop is not to remove human judgement but to make it compound — every correction makes the next run a little better.

## Citable facts

- The stated goal of an agent feedback loop is not to remove human judgement/taste but to make it *compound* — every correction makes the next run a little better; the agent becomes "a working memory of how the team thinks" ([[wiki/sources/petradonka-agents-need-feedback-loops]]).

## Related concepts

- [[self-annealing]] — the mechanism by which corrections accrete into durable improvement.
- [[human-in-the-loop]] — the gate that captures each human correction.
