# Agent skills

> Reusable, packaged agent workflows (e.g. Claude Code `/`-commands) that encode a procedure an agent can invoke. Appears in this wiki via [[wiki/sources/nurijanian-goal-for-product-managers]].

## Definition

This is a **stub**. Agent skills, as they appear in [[wiki/sources/nurijanian-goal-for-product-managers]], are packaged reusable workflows authored for an agent and invoked as commands — exemplified by [[wiki/entities/matt-pocock|Matt Pocock]]'s viral `/grill-me` skill and [[wiki/entities/ryan-singer|Ryan Singer]]'s `/shaping` skill, bundled together into a product-manager workflow for writing requirements with AI.

## Treatment across sources

- [[wiki/sources/nurijanian-goal-for-product-managers]] bundles third-party skills (`/grill-me`, `/shaping`) into a PM OS for requirement-writing, treating skills as composable, shareable workflow units hosted on GitHub.
