# Sequential thinking

> A structured step-by-step reasoning pattern — break a problem into steps, evaluate alternatives, debug logically — framed as a distinct agent capability. Appears in this wiki via [[wiki/sources/suryanshti777-9-claude-code-plugins-senior-engineer]]. (The reasoning pattern; the specific MCP server implementing it is catalogued separately as the sequential-thinking-mcp entity.)

## Treatment across sources

- [[wiki/sources/suryanshti777-9-claude-code-plugins-senior-engineer]] frames it (Tool 5) as structured step-by-step reasoning (break into steps, evaluate alternatives, debug logically) — a distinct capability that improves system design and debugging.

## Sub-claims and details
